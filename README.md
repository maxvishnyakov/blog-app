### 4. Приложение доступно по адресу localhost:8080

## API Эндпоинты
- Получение списка постов (на главной странице — ленте постов) — метод GET `/api/posts?search=Lalala&pageNumber=1&pageSize=5`, где search — это строка из поля поиска, pageNumber — номер текущей страницы, pageSize — число постов на странице (все поля обязательные)
- Получение поста (на странице поста) — метод POST: `/api/posts/1`, где 1 — идентификатор поста
- Добавление поста (на главной странице — ленте постов) — метод POST: `/api/posts`
- Редактирование поста (на странице поста) — метод PUT: `/api/posts/3`
- Удаление поста (на странице поста) — метод DELETE: `/api/posts/1`, где 1 — идентификатор поста
- Инкремент числа лайков поста (на странице поста) — метод POST: `/api/posts/1/likes`, где 1 — идентификатор поста
- Обновление картинки поста (при добавлении и редактировании поста) — метод PUT: `/api/posts/1/image`, где 1 — идентификатор поста
- Получение картинки поста (главная страница — лента постов и страница поста) — метод GET: `/api/posts/1/image`, где 1 — идентификатор поста
- Получение комментариев поста (на странице поста) — метод GET: `/api/posts/1/comments`, где 1 — идентификатор поста
- Получение комментария поста (на странице поста) — метод GET: `/api/posts/1/comments/2`, где 1 — идентификатор поста, 2 — идентификатор комментария
- Добавление комментария к посту (на странице поста) — метод POST: `/api/posts/1/comments`, где 1 — идентификатор поста
- Редактирование комментария к посту (страница поста) — метод PUT: `/api/posts/1/comments/2`, где 1 — идентификатор поста, 2 — идентификатор комментария
- Удаление комментария (страница поста) — метод DELETE: `/api/posts/1/comments/2`, где 1 — идентификатор поста, 2 — идентификатор комментария
